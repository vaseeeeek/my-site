CLI - работа с консолью

Папки
08.11.2022

  • cd - переход в папку с указанием пути

    ls, ls -l, ls -la(-l -a) показать все файлы (-l распечатать списком, -la также распечатать системные файлы)

    pwd - распечатать путь до папки в которой сейчас нахожусь

    mkdir one - создать папку

    mkdir two three four - создание нескольких папок

    mkdir -p img/cover/test - -p создает все недостающие папки

    mkdir -p {img/cover/test,css/loyout,script/modals} - создает несколько папок с своими вложенностями

    reder new удаление папки new (не работает ели в папке остались файлы)

    rm -r new удаление папки и всех файлов внутри

    cp -r one two - скопировать one в папку two

    mv -r one two - переместить one в папку two

Файлы
08.11.2022

  • touch text.txt - создание файла

    rm text.txt - удаление файла

    mv index.html test - вырезать index.html в папку test

    mv index.html new.html - переименует файл

    cp index.html test - скопировать index.html в папку test

    find . -iname "*.html" поиск файлов в директории по имень *.html (-i не обращать внимание на регистр)
